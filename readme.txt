This plugin allows you to integrate Beevi's sport club management solution with your Wordpress site.

== Description ==

Beevi is a complete solution to manage your sport club or organization.  

*  Communicate with your members
*  Manage your club's membership
*  Engage the conversation with your members

This plugin allows you to integrate Beevi with your Wordpress site.



== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin file to the `/wp-content/plugins/` directory
1. Unzip it
1. Activate the plugin through the 'Plugins' menu in WordPress
1. On the Wordpress administration menu, click on Beevi and insert your Beevi API key
1. Now, go to the widget section.  You can now add the Beevi widget to your Wordpress site

== Frequently Asked Questions ==

= Where can I find my Beevi API key? =

1. Login into your Beevi account.
1. Select the Organization tab.
1. If you selected "Wordpress" as your website type, your API key will appear there
